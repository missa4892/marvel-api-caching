# Marvel Charater API

## Description
Uses 
[Nest](https://github.com/nestjs/nest) framework TypeScript repository.

## Installation

```bash
$ yarn
```

## Running the app

```bash
$ yarn start
```
Then access the server at `http://localhost:8080`

## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```

## Swagger

```bash
$ yarn start
```

Then proceed to `http://localhost:8080/api`
